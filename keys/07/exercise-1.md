## Chapter ["Organising information: unordered structures"](https://comp-think.github.io/book/07.pdf), exercise 1

### Text
Write a code in Python to create a set of the following elements: `"​Bilbo"`, `"​Frodo"`, `"​Sam"`, `"​Pippin"`, `"​Merry"`. 

### Answer
```python
my_set = set()
my_set.add("Bilbo")
my_set.add("Frodo")
my_set.add("Sam")
my_set.add("Pippin")
my_set.add("Merry")
```
